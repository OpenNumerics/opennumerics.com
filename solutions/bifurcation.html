PyCont: Continuation and Bifurcation Detection for Complex Systems

The Problem

Numerical continuation and bifurcation detection is one of the most important computational tools from the 20th century. It is a workhorse in all of computational science and engineering. Continuation is used to understand how complex systems change as parameters vary. It is fundamental to studing highly complex dynamical systems and their transition points, such as bistability in chemical networks, critical transitions in mechanical systems, pattern formation in PDEs, stable conformations of large molecules and some problems in aerodynamics.

However, the only robust tool for continuation and bifurcation detection is Auto. It was written in Fortran in the 1980s and the whole world still relies on it. modern and easy-to-use implementation in a high-level language exists. As a result, there is no way to couple it to modern software, enable GPU acceleration through CUDA, or link it to high-performance computing environments. This impactful algorithm has no way to grow.

Our Approach

At Open Numerics, we've developed PyCont-Lite, the first modern, lightweight and scalable Python API for numerical continuation and bifurcation detection. PyCont-Lite supports all modern tensor libraries (numpy, jax, pytorch) and also has full GPU support enabled through CUDA. It uses more advanced numerical routines and solvers than older packages that have not been updated.

PyCont is built around a fast and robust Newton-Krylov solver that efficiently solves large-scale nonlinear systems, including those arising from PDE discretizations. It can, for example, also be used to trace the classification threshold of neural networks, and thousands of other use cases we haven't thought about. As a bonus, PyCont-Lite can detect bifurcation points and branch-switch automatically.

The Result

PyCont-Lite achieves similar accuracy to older continuation packages, yet is faster, smaller and allows for GPU acceleration. It provides a fast, flexible, and extensible platform, and its clean Python interface integrates easily with scientific codebases and simulation pipelines, enabling deeper insight into your models.

PyCont-Lite can be licensed. An older open source version is available on GitHub as well.
